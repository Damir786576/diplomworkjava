# Дипломный проект — Клон Авито (SkyPro)

Полноценное веб-приложение подобие Авито, написанное на Spring Boot.

## Что есть в проекте
- Регистрация и авторизация 
- Создание, редактирование, удаление объявлений
- Загрузка фото к объявлениям
- Комментарии к объявлениям
- Личный кабинет с аватаркой пользователя
- Поиск и просмотр всех объявлений

## Структура 
**controller**              | REST-контроллеры — точки входа всех запросов от фронтенда <br>• **AdsController** — операции с объявлениями (создание, редактирование, удаление, получение списка и отдельного объявления)<br>• **UserController** — работа с профилем пользователя (данные, смена пароля, загрузка аватарки)<br>• **AdCommentController** — создание, редактирование и удаление комментариев к объявлениям<br>• **AuthController** — регистрация и аутентификация пользователей<br>• **ImageController** — отдача изображений объявлений и аватарок по прямым ссылкам |

| **service**               | Бизнес-логика приложения <br>• **AdService** — вся работа с объявлениями и их изображениями<br>• **UserService** — управление данными пользователя и аватаркой<br>• **CommentService** — операции с комментариями<br>• **AuthService** — регистрация и проверка учётных данных<br>• **UserDetailsServiceImpl** — интеграция со Spring Security |

| **mapper**                | Преобразование между сущностями базы данных и DTO <br>Скрывает внутренние поля (например, пароль) и формирует корректные ссылки на изображения |

| **model**                 | JPA-сущности — прямое отображение таблиц БД <br>• **User** — пользователи<br>• **Ad** — объявления<br>• **AdComment** — комментарии |

| **repository**            | Интерфейсы Spring Data JPA <br>Обеспечивают доступ к данным (поиск, сохранение, удаление) |

| **dto.request**           | DTO для входящих запросов от клиента |

| **dto.response**          | DTO для ответов клиенту (без лишних и служебных полей) |

| **config**                | **WebSecurityConfig** — настройка Spring Security, CORS, Basic Auth и правил доступа к эндпоинтам |

| **filter**                | Дополнительные фильтры (например, корректная передача CORS-заголовков при Basic Auth) |

| **uploads/**              | Хранилище загруженных изображений <br>• `uploads/ads/` — фотографии объявлений<br>• `uploads/users/` — аватарки пользователей |

## Как запустить проект?

- Открыть Docker
- В терминале написать  docker run -p 3000:3000 --rm ghcr.io/dmitry-bizin/front-react-avito:v1.21
- Открыть HomeworkApplication и нажать на run
- Перейти в браузер и открыть http://localhost:3000/
